# Security Policy

## Passwords and MFA

All accounts must use the company password manager. Multi-factor authentication
is mandatory for email, the code repository and the cloud console.

## Devices

Laptops must have full-disk encryption enabled and the screen must lock after
5 minutes of inactivity. Report a lost or stolen device to security@northwind
within 24 hours.

## Data handling

Customer data may only be stored in approved systems. Never paste customer data
into external tools or third-party chatbots. Production database access requires
an approved ticket and is granted for a maximum of 7 days at a time.
